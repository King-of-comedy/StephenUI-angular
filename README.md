# stephenUi-angular
## 安装与使用

```
// 注： 项目依赖angular1.x版本
npm install stephen-ui-angular
```

## 引入方式

### ES6

```
import stephenUi from '../node_modules/stephen-ui-angular';

angular.module('app', [stephenUi]);
```

### ES5

```
<script src="../node_modules/stephen-ui-angular/build/stephen-ui.js"></script>
```
